### 功能点总结清单

| 类别         | 功能点       | 核心描述                                                     |
| :----------- | :----------- | :----------------------------------------------------------- |
| **播放控制** | 基础播放控制 | 播放、暂停、停止                                             |
|              | 精确步进控制 | 切换帧(K)/秒(S)模式，用A/D键控制                             |
|              | 音量控制     | 快捷键或进度条调节                                           |
|              | 播放模式     | 随机播放、最新优先                                           |
|              | 视频旋转     | 旋转90°\|180°\|270°并保存状态                                |
|              | 跳帧播放模式 | 等间隔快速预览视频内容                                       |
|              | 截图轨道     | 可视化时间轴，点击截图跳转到视频对应时间点                   |
|              | 下个视频     |                                                              |
| **截图**     | 原分辨率截图 | 按E键保存当前帧                                              |
|              | 截图旋转设置 | 独立设置输出截图的旋转角度                                   |
|              | 导出功能     | 导出指定视频的截图文件夹到指定目录                           |
| **文件定位** | 点赞功能     | 点击心形图标记录喜好                                         |
|              | 点赞页面     | 按点赞数排序浏览高价值视频                                   |
|              | 收藏功能     | 标记重要视频                                                 |
|              | 收藏列表     | 查看所有收藏的视频                                           |
|              | 标签系统     | 创建图片/文本标签并分配给视频                                |
|              | 标签搜索     | 使用标签快速定位的被标记过的视频                             |
|              | 最近播放列表 | 查看和切换最近看过的50个视频                                 |
| **文件管理** | （软）删除   | 移动文件至待删除区，需确认                                   |
|              | 刷新功能     | 重新扫描以发现新文件，被重命名或移动文件被重新绑定，被外部编辑内容或删除清理伴生数据 |
| **视频编辑** | 智能视频裁剪 | 标记并删除无用片段，保留精华                                 |
|              |              |                                                              |
